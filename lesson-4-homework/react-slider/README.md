# hw
a plug-in unit of react as an carousel component.
一个React.js 的轮播组件，可以直接使用，支持配置
# 本地 demo（使用 webpack 来管理的依赖）
```
npm install -d
// 启动本地服务器
npm run server
// 开发环境打包依赖 ， 生产环境是 npm run dist
npm run build
```
Open up http://localhost:3000/ in your browser.
在浏览器打开 http://localhost:3000/。
打包的文件在 ``dist`` 下面。
